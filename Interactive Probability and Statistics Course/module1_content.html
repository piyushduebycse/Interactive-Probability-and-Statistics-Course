1.1 Introduction to Probability

Probability is the branch of mathematics that deals with the study of randomness and uncertainty. It provides a framework for modeling and analyzing random phenomena, making it an essential tool in various fields such as statistics, science, engineering, finance, and everyday decision-making.

The concept of probability emerged from the study of games of chance in the 17th century. Mathematicians like Blaise Pascal and Pierre de Fermat developed the early foundations of probability theory while analyzing gambling problems. Today, probability theory has evolved into a sophisticated mathematical discipline with applications far beyond its origins.

What is Probability?

Probability is a measure of the likelihood that an event will occur. It is expressed as a number between 0 and 1, where:

  • A probability of 0 indicates impossibility (the event will not occur)
  • A probability of 1 indicates certainty (the event will occur)
  • A probability between 0 and 1 indicates the degree of likelihood (higher values indicate more likely events)

There are three main interpretations of probability:

  1. Classical (theoretical) probability: Based on equally likely outcomes. If an experiment has n equally likely outcomes and m of these outcomes correspond to event A, then P(A) = m/n.
  2. Relative frequency (empirical) probability: Based on the observed frequency of an event. If an experiment is repeated N times and event A occurs m times, then P(A) ≈ m/N as N becomes very large.
  3. Subjective probability: Based on personal judgment or belief about the likelihood of an event, often used when classical or frequency approaches are not applicable.

Example: Different Probability Approaches

Classical approach: When rolling a fair six-sided die, the probability of getting a 3 is 1/6 because there are six equally likely outcomes, and only one of them is a 3.

Frequency approach: If you roll a die 600 times and observe that a 3 appears 98 times, the empirical probability would be 98/600 ≈ 0.163.

Subjective approach: A weather forecaster might say there's a 70% chance of rain tomorrow based on their expertise and analysis of weather patterns.

1.2 Sample Spaces, Events, and Outcomes

To formalize probability, we need to define some fundamental concepts:

Sample Space

The sample space (denoted by S or Ω) is the set of all possible outcomes of a random experiment. Each outcome in the sample space is a complete, detailed result of the experiment.

Event

An event is a subset of the sample space, representing a collection of outcomes. Events are typically denoted by capital letters (A, B, C, etc.).

Outcome

An outcome is a single result of an experiment. It is an element of the sample space.

Example: Rolling a Die

Sample space: S = {1, 2, 3, 4, 5, 6}

Event A: Getting an even number = {2, 4, 6}

Event B: Getting a number greater than 4 = {5, 6}

Outcome: Rolling a 3 is a single outcome in the sample space

Events can be combined using set operations:

  • Union (A ∪ B): The event that either A or B (or both) occur
  • Intersection (A ∩ B): The event that both A and B occur
  • Complement (A'): The event that A does not occur

Example: Set Operations with Events

Using the die example above:

A ∪ B: Getting an even number or a number greater than 4 = {2, 4, 5, 6}

A ∩ B: Getting an even number greater than 4 = {6}

A': Not getting an even number = {1, 3, 5}

1.3 Probability Axioms and Rules

Probability theory is built on three fundamental axioms, proposed by Andrey Kolmogorov in 1933:

Axiom 1: For any event A, P(A) ≥ 0

Axiom 2: P(S) = 1, where S is the sample space

Axiom 3: For mutually exclusive events A₁, A₂, A₃, ..., P(A₁ ∪ A₂ ∪ A₃ ∪ ...) = P(A₁) + P(A₂) + P(A₃) + ...

From these axioms, we can derive several important probability rules:

Complement Rule: P(A') = 1 - P(A)

Addition Rule: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Multiplication Rule: P(A ∩ B) = P(A) × P(B|A) = P(B) × P(A|B)

Example: Applying Probability Rules

In a standard deck of 52 cards, what is the probability of drawing either a heart or a face card?

Let A = drawing a heart, and B = drawing a face card.

P(A) = 13/52 = 1/4 (there are 13 hearts in a deck)

P(B) = 12/52 = 3/13 (there are 12 face cards: jacks, queens, and kings)

P(A ∩ B) = 3/52 (there are 3 face cards that are hearts: jack, queen, and king of hearts)

Using the addition rule: P(A ∪ B) = P(A) + P(B) - P(A ∩ B) = 1/4 + 3/13 - 3/52 = 13/52 + 12/52 - 3/52 = 22/52 = 11/26

1.4 Conditional Probability and Independence

Conditional Probability

Conditional probability is the probability of an event occurring given that another event has already occurred. It is denoted as P(A|B), read as "the probability of A given B."

Conditional Probability Formula:

P(A|B) = P(A ∩ B) / P(B), where P(B) > 0

Example: Conditional Probability

In a class of 30 students, 18 are taking mathematics, 15 are taking physics, and 10 are taking both subjects.

If a student is selected at random, what is the probability that the student is taking mathematics given that they are taking physics?

Let M = student takes mathematics, and P = student takes physics.

We want to find P(M|P).

P(M ∩ P) = 10/30 = 1/3 (probability of taking both subjects)

P(P) = 15/30 = 1/2 (probability of taking physics)

P(M|P) = P(M ∩ P) / P(P) = (1/3) / (1/2) = 2/3

So, the probability that a student takes mathematics given that they take physics is 2/3.

Independence

Two events A and B are independent if the occurrence of one does not affect the probability of the other. Mathematically, A and B are independent if and only if:

Independence Condition:

P(A ∩ B) = P(A) × P(B)

Equivalently, P(A|B) = P(A) or P(B|A) = P(B)

Example: Independence

When flipping a fair coin twice, let A = getting heads on the first flip, and B = getting heads on the second flip.

P(A) = 1/2 and P(B) = 1/2

P(A ∩ B) = 1/4 (probability of getting heads on both flips)

Since P(A ∩ B) = P(A) × P(B) = 1/2 × 1/2 = 1/4, events A and B are independent.

1.5 Bayes' Theorem

Bayes' theorem provides a way to update probabilities based on new evidence. It is a fundamental result in probability theory with wide-ranging applications.

Bayes' Theorem:

P(A|B) = [P(B|A) × P(A)] / P(B)

Where P(B) = P(B|A) × P(A) + P(B|A') × P(A')
